## Instalação:
Passo 1: 
Configure a conexão do banco de dados.
- API: "appsettings.json" 
```json
{
  "ConnectionStrings": 
    {
      "Database": "", 
    }
}
```

Passo 2:   
Com a IDE Visual Studio ou Visual Studio Code: File > Open project. E execute o projeto com docker.
